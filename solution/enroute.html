Solution: En Route
Answer: BRABANCONNE

Written by ev

The appearance of the example image, in conjunction with the title, should remind us of a rapid transit map. Each image clues the name of a rapid transit station; each pair of stations is on the same line and network.

We should then notice that the example image has a letter on each station, and they spell out "EXAMPLE", the same as the only named station. This clues that for each pair, we should find a station en route that has the same number of letters as there are stations on the route (including the termini). These are unique for each pair. For example, the route for the first pair (Bank - Mile End) has four stations (Bank, Liverpool Street, Bethnal Green, Mile End), and the only station there that is four letters long is BANK.

We then have a correspondence between stations and letters, and for each pair we extract the letter corresponding to the station with the same length as the route. For Bank - Mile End, B corresponds to Bank, A corresponds to Liverpool Street, N corresponds to Bethnal Green, and K corresponds to Mile End, so we extract B.

The original ordering has the extraction stations in alphabetical order. The route lengths are unique and consecutive, so we should reorder the pairs in order of increasing length from 4 to 14. The extracted letters then spell out the answer, BRABANCONNE.
