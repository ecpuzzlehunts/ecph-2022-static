Solution: Cobble
Answer: ZLOTY

Written by rubyleehs

This puzzle is based on Wordle (maybe you've heard of it before? it seems popular or something), but unlike regular Wordle, you can't enter any guesses, and the patterns of correct letters are already given. Each set seems terribly underconstrained until we make a key realization - all the words in a set belong to the same category, clued by the ???. In order to make this clear for solvers, we start with a couple easy categories: GREEK LETTERS, followed by the NATO ALPHABET. As suggested by the rotated "=" signs, the last word in each set is the first word in the next. The completed grid is as follows:

Author’s Notes

Originally, we wanted to avoid giving any category clues. Three variants of this puzzle were made, but testsolvers found some categories unreasonable or un-fun to get without extra clues. We would love to see someone make a variant of this without anything but the initial starting word. In theory, it's possible to omit the starting word too, but construction would be even more constrained.

Additionally, the final set of the puzzle was originally a "failed" Wordle, with all six guesses used. However, external test solves showed that it gives the idea that the answer is NOT a currency. After some discussion and a vote, we decided it was better to minorly nerf the puzzle to be as straightforward as possible, considering it was one of three puzzles unlocked at the start.

It is possible to figure out which currency was removed, so I leave that as a challenge to the reader. It is a currently circulating currency and has to be uniquely identifiable.
