Solution: On Thin Ice
Answer: SEWING MACHINE

Written by chimpaznee

We are presented with instructions to spell out strings of letters by moving on a set of grids that contain ice and snow. However, these strings of letters are not all present on the grids' initial state. From the “title” of each grid and the given letters, we should find out a way to fill each of the grids with letters so that it follows the clued pattern. If this is done correctly, for each grid there is a unique path that, when followed according to the instructions, spells out the requested string.

Final Exercise

This is an icebarn logic puzzle, except that the arrow directions need to be borrowed from elsewhere (in this case, from the paths we've made in the ice skating exercise grids). In addition, some letters should appear on the ice, presumably also from same 8 grids.

When we lay out all of the paths we've previously made in the ice over the icebarn grid, we can observe that each one travels over exactly one icebarn, which means we will have to borrow the letters from its respective grid and put them there. Author’s Notes

This was one of the first puzzles written for this hunt. It went through several iterations, though the core idea of “figure out how to fill a grid with letters” stayed. The initial version had rocks instead of snow, and you had to figure out the paths from 8 given starting locations and a pair of given directions. After testsolver feedback that this version did not quite deliver, I ended up reworking the puzzle from the ground up to include the final icebarn puzzle, and use the paths in a more interesting way.

Thank you quatrevingtneuf for giving me some ideas for the letter grids (namely, flavor text, US states and Pascal’s triangle).
