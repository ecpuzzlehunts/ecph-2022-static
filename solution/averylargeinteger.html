Solution: A Very Large Integer
Answer: SOCOTRA

Written by chimpaznee

This puzzle is entirely contained in a single, very large integer. But how do we break into it? We can start by looking for clues near the beginning of the number. If its first digits are split up into groups of 2, each such group is a number 01-26, which means we can translate these to letters. Doing this, the first 228 digits spell out a message, which reads like this once spaces are added:

SUPERB START THE FIRST DIGIT IS FOUND IN HONEY AND NOW ISOLATE THE FOLLOWING DIGITS IN CHUNKS OF ONE HUNDRED AND USE RAPIDTABLES TO CONVERT TO BASE THIRTY SIX

This message has two main parts. The first one states that “the first digit is found in honey”. We do not know the context of this yet, but we can observe that the digit ONE is found as a substring of honey. The second part is an instruction. If we isolate the digits that follow the message in blocks of 100, and convert each block to base 36, using the base converter on rapidtables, we can get 10 blocks of characters (with the last 93 digits only base36'ing to gibberish):

As stated above, "HONEY" contains the substring ONE. The first digit is 1.

Conclusion

To obtain the final answer, we will need to treat the minipuzzles' answer digits as geographic coordinates. For that, both the latitude and longitude values will be positive. We'll group the digits in two sets of three, and have one digit after the decimal mark in each group. The resulting coordinates are 12.5, 53.9. These coordinates land at the very center of the island of SOCOTRA! With the information that the final answer is 7 letters long, this is the correct answer to the puzzle.

Author’s Notes

I thought it would be funny to write a puzzle that is contained entirely in a single number.

This puzzle had to be rewritten less than one month before the hunt started. The previous answer to it was QUBATY TOWER. While the structure was the same (6 minipuzzles, each of them extracting to a digit), the final step involved Qubaty Tower's Emporis Building Number of 105362, which seemed like a reliable, unchanging source that is relevant to the answer. Or so we thought. On Sep. 14, Emporis and its entire database were shut down. This meant that we had to give the puzzle a major revision. We had a second look at the meta, and luckily quatrevingtneuf gave me a much nicer answer to work with, SOCOTRA. Luckily, the coordinates 12.5, 53.9 land comfortably in the middle of the island, which means that I could keep the structure of six minipuzzles, where each of them has a digit as an answer. The minipuzzles were all there, I've just had to change their extracted numbers and reorder them.

Because the final step involves coordinates, the extracted digits have differing importance. I tried to put the minipuzzles that I thought are more interesting or difficult as the first or second digits of the latitude or longitude, to encourage more teams to solve them. The very first digit, however, is given for free. This was also the case when I was working with the answer QUBATY TOWER, and I did not want to change this.

I like this type of puzzle that isn't what it looks like on the surface, though I've found that a lot of these puzzles are on the high end of the difficulty scale, so I wanted to make a much more approachable version of the type. I drew some inspiration from a puzzle in the 2022 Mystery Hunt, Lists of Large Integers, including the puzzle title.
