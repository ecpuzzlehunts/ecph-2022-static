Solution: Much Assembly Required
Answer: PSEUDONYMS

Written by SeptaCube

There is a list of Across and Down entries, but they are all gibberish. We can try to treat this as a fill-in, but there is no diagram, and there are even a few unknown entries! So how do we get a start?

We can count the number of letters in all entries. Without the ???? entries, the letter count of the Down entries is lower than the Across entries by 16, which means that it is likely that each ???? entry is four letters long. Also, there is only one Across entry that is eight letters long. By assuming that the resulting grid should be symmetrical (like a regular crossword grid), the entry "SAOYRRWC" must be in the centre. This entry also has the letter Y, which only appears in the Down entry "YILP", so we can make them cross, and black out 6 cells (ends of SAOYRRWC, ends of YILP, and the 180-degree rotation of the latter).

From here, we can keep building the rest of the grid, starting from the centre and going outwards. There is a unique way to do this, even with the presence of 8 unknowns:

The unknown entries are actually English words. By starting from the top-right and following the path, these read out an instruction with two parts: MAKE GRID INTO CUBE NETS; SAME FACE MEET.

For the first part, MAKE GRID INTO CUBE NETS, we will need to find a way to divide the grid into cube nets. Again, there is a unique way to do this. For this step, it is useful to count the number of cube nets the solution would have. That number is 27. Because it is an odd number, this means that there will be a cube net at the very centre. We can start from it and work our way outwards. Like in the fill-in step, the cube net division has rotational symmetry (the colour choices in this image are arbitrary):

For the second part, SAME FACE MEET, we will need to assemble something with our cubes, such that faces with the same letter meet. The number of nets (27) suggests building a 3x3x3 cube. A daunting task, but it is nevertheless possible. There are several ways to approach this, including modelling it on sheets, coding out the logic, or recreating the cubes on graph paper.

A step-by-step walkthrough is available at the Appendix, but a key realization to get started is that one of the cubes has the letters FJLQXZ. Each of these letters only has one other cube face to pair up with. Three of these cubes have a letter that only appears once (H/U/Y) on their opposite face, with all 3 cubes meeting at a corner. That means the FJLQXZ cube has to be at the centre.

From there, we can complete the 3x3x3 cube in a unique way. This is the result (3 of the faces of the completed cube, and its net):


Once we finish assembling the cube, we can play around with it, and look to see if it reads out anything. Starting along one of the edges, we can read "MARK TWAIN". We keep reading in the same direction to finish the cluephrase: MARK TWAIN, DR SEUSS AND MARY WESTMACOTT (TEN). These are all examples of PSEUDONYMS.
